### Atividade da aula 3
Durante a aula 3 de Cross-Platform Application Development, aprendemos sobre JSX (extensão do
Javascript para React Native e sobre os componentes core. Esta atividade foi proposta para 
personalizarmos o exemplo passsado durante a aula, com uma foto de perfil fictícia, nome, 
turma, semestre e curso que o aluno está fazendo. Também foi exigido dois links no app, 
apesar de não precisarem estar funcionais, por enquanto.
